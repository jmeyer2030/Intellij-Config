Instructions for mirroring my IntelliJ IDEA Setup:

1. Download IntelliJ IDEA Community Edition
 - https://www.jetbrains.com/idea/download/?section=windows
2. Replace the plugins folder with the one in this repo
 - C:\Program Files\JetBrains\IntelliJ IDEA Community Edition 2024.3.1\plugins
3. Import the settings folder from within IntelliJ IDEA
 - file -> Manage IDE Settings -> Import Settings
4. copy .ideavimrc to your user folder
 - C:\Users\[user]
